# GiardDesign — zadanie rekrutacyjne

Responsywne wdrożenie landing page’a przygotowane na podstawie projektu Figma w procesie rekrutacyjnym dla agencji [adRespect.pl](https://adrespect.pl/).

## Zakres

- wierne odwzorowanie widoku desktopowego,
- autorski układ na tablet i urządzenia mobilne,
- rozwijana galeria i mobilna nawigacja,
- animacje wejścia, mikrointerakcje oraz obsługa `prefers-reduced-motion`,
- semantyczny HTML i dostępne etykiety elementów interaktywnych,
- zoptymalizowane lokalne zdjęcia z projektu.

## Technologie

- Vue 3 Composition API,
- Vite i TypeScript,
- Tailwind CSS 4.

## Uruchomienie

Wymagany jest Node.js 22.13 lub nowszy.

```bash
npm install
npm run dev
```

Weryfikacja wersji produkcyjnej:

```bash
npm test
```

Gotowe pliki do wysłania na hosting znajdują się w katalogu `dist`.

Kod strony jest częścią zadania rekrutacyjnego. Projekt graficzny pozostaje własnością adRespect.pl.
